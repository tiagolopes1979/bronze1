# Pedito Bronze

Landing page estática para divulgação de serviços de bronzeamento, agendamento via WhatsApp e vitrine de produtos do segmento.

## Estrutura

- `index.html`: estrutura principal da página
- `styles.css`: estilos visuais e responsividade
- `script.js`: renderização dinâmica de serviços e produtos, validação do formulário e interações da navegação

## Funcionalidades

- Hero comercial com foco em conversão
- Serviços renderizados por dados em `script.js`
- Vitrine de produtos com fotos e links externos
- Menu responsivo com hover e abertura mobile
- Formulário com validação e envio formatado para WhatsApp
- Seções de prova social, FAQ e oferta comercial

## Publicação

O projeto está preparado para deploy automático no GitHub Pages via GitHub Actions.
